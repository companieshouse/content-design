secretary

Lower case.


security code

This is a code that is sent to a user as part of the 2 factor authentication process when accessing a service. It is sent either by text message or authenticator app.

Never use:

  • 2-factor authentication
  • 2 step verification
  • one time code
  • OTC
  • one time passcode
  • OTP
  • confirmation code

service address

Lower case.


Service names

Always write out service names in full, never abbreviate. Use upper case for first letter of the first word, the rest lower case, unless it’s a proper noun.

Standard Industrial Classification (SIC) code

Title case for the first letter of each word, except 'code'. Explain the acronym at first use, then abbreviate to SIC code.


strike off

Lower case.

Always two separate words, never hyphenated. For example ‘To strike off a company’, ‘Strike off action’ and ‘This company has been struck off’.
